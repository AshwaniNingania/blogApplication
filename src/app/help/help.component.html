Blog Application

In this Application, we are having following pages :

1. Login Page :

Sign Up : Here You can create new user with a unique username and a valid password by signup button.

Login : With your userId and password you can login. After login page automatically redirected to Blogs Application.

2. Blogs Application page :

New Blog : Add a new Blog with a title, category, and a description.

Blogs : Here we can see all user's blogs but we can edit or delete own blogs only.
A blog having the title, like option, time of uploaded, category, description and the author name.

Filter Blogs : You can filter the blogs by blog title and category.
we can also see our favourites blogs only by clicking on favourites blogs button.
